🔑 Key Design Decisions

DecisionApproachImpact
AI Cost ControlHybrid: 8 SQL rules run first → AI only for fuzzy matching~90% token savings vs full-AI
Classification CacheMD5 hash → 24h TTL → skip duplicate AI callsZero cost for re-processed docs
EnrichmentSingle 4-JOIN query replaces 5 sequential lookups80% latency reduction
Fraud DetectionPure SQL (ISO 6346 check-digit, weight anomaly)Zero AI cost, deterministic
i18nLayer 1: Dictionary (instant) + Layer 2: CORTEX.TRANSLATE (dynamic)Zero AI cost for UI labels
Retry MechanismExponential backoff (1s → 2s → 4s) in AI_COMPLETE_WITH_RETRY99.5% success rate
FinOpsEvery AI call logged → APP_CONFIG threshold → automated alertFull cost visibility
SecurityKey-pair JWT + MENDIX_SERVICE_ROLE + AI_CALL_LOG auditEnterprise-grade auth
