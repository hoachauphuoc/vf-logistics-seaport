Feature 1: Auto-Classification

17 Maritime Document Types

CALL CLASSIFY_DOCUMENT_TEXT('BILL OF LADING No. MAEU123...');
→ {document_type: "BILL_OF_LADING", confidence: 0.95}
  • B/L, Invoice, Packing List, Certificate of Origin, DG Declaration, Manifest, EDI, +10 more
  • Confidence scoring — human-in-the-loop for <85%
  • Retry mechanism — exponential backoff (1s, 2s, 4s)
  • All calls logged to AI_CALL_LOG for audit + cost tracking
Speaker Notes: Classification is step one. Documents below 85% confidence get flagged for human review. Retry mechanism ensures reliability under load.
6/16

Feature 2: Hybrid Cross-Check

Rule-Based + AI (saves 90% tokens)

8 SQL Rules (FREE)

  • Weight discrepancy (>2%)
  • Package count mismatch
  • Vessel name mismatch
  • Voyage number mismatch
  • ETD date mismatch
  • Incoterms mismatch
  • Volume/CBM discrepancy
  • Container number mismatch

AI Fuzzy Match (edge cases)

  • "VN SEAFOOD JSC" = "VIETNAM SEAFOOD JOINT STOCK COMPANY" ✓
  • Only invoked when party names differ
  • ~90% cost reduction vs full-AI
Speaker Notes: Most innovative feature. 8 SQL rules run first at zero cost. AI only for party name matching — requires semantic understanding of abbreviations.
