Step 1: Auto-Classification

17 Document Types, 95%+ Confidence

CALL CLASSIFY_DOCUMENT_TEXT('BILL OF LADING No. MAEU123...');
→ {document_type: "BILL_OF_LADING", confidence: 0.95, cached: false}
  • MD5 Classification Cache — 24h TTL, zero cost for re-processed docs
  • Confidence < 85% → human review queue (HITL)
  • Retry with exponential backoff (1s, 2s, 4s)
  • All calls → AI_CALL_LOG (audit + cost tracking)

ACTION: Routes document to correct processing pipeline.

Step 2: Hybrid Cross-Check

8 SQL Rules (FREE) + AI Fuzzy Match (edge cases only)

  • Vessel name mismatch
  • Voyage number mismatch
  • ETD date mismatch
  • Incoterms mismatch
  • Volume/CBM discrepancy
  • Container number mismatch

AI (only when needed)
